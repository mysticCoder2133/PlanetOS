# PlanetOS

A website that raises awareness about climate change and helps users understand their environmental impact.

## Description

PlanetOS explains one of the biggest problems the world is facing today: climate change.  
It features immersive, scroll-based environments (ReefOS, IceOS, ForestOS) that educate users about different environmental issues. The website also includes an interactive Carbon Footprint Calculator, allowing users to estimate their impact and reflect on possible solutions.
Live Demo: https://mysticcoder2133.github.io/F32/
